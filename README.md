# Loan Risk Analysis using Exploratory Data Analysis

## Overview

This project analyzes historical loan data to identify patterns associated with loan defaults and borrower risk.

## Objectives

- Clean and preprocess loan data
- Detect and handle missing values and outliers
- Perform exploratory data analysis
- Identify key risk factors associated with default
- Generate business recommendations

## Key Findings

- Higher interest rates are associated with increased default risk.
- Lower credit grades exhibit higher charge-off rates.
- Debt consolidation is the most common loan purpose.
- Certain borrower segments contribute disproportionately to loan losses.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
